▸ 흥분 전도·막전위 파트
함정 ① 시냅스 전달 방향 6/6회 반복
정상 개념
시냅스 전달은 축삭돌기 말단 → 다음 뉴런 가지돌기 방향으로만 가능. 역방향 전달은 구조적으로 불가.
오답 유도 내용
"㉠에 자극을 주면 시냅스 이전 방향으로도 흥분이 전달된다" → 오답
왜 틀렸나
수용체가 시냅스 후 뉴런에만 존재. 신경전달물질이 이전 방향으로 작용할 수용체가 없음.
함정 ② -80mV의 상태 혼동 6/6회 반복
정상 개념
−70mV = 휴지전위
−80mV = 과분극 (K⁺ 과잉 유출)
+30mV = 탈분극 최고점
오답 유도 내용
"−80mV이므로 휴지 상태이다" → 오답

"−80mV일 때 안정 상태이다" → 오답
왜 틀렸나
휴지전위는 정확히 −70mV. −80mV는 K⁺ 과잉 유출된 과분극. 휴지 상태가 아님.
함정 ③ 전도 속도 계산 시 1ms 빼기 누락 5/6회 반복
정상 개념
속도 = 거리 ÷ (경과시간 − 막전위 변화까지 걸린 시간)

예) 4ms 경과, +30mV → 이동시간 = 4−1 = 3ms
오답 유도 내용
1ms를 빼지 않고 4ms로 그대로 나눔
→ 6cm ÷ 4ms = 1.5cm/ms → 오답
왜 틀렸나
+30mV 도달까지 약 1ms 소요됨. 경과시간에서 반드시 빼야 이동 시간 산출 가능. 정답: 6cm ÷ 3ms = 2cm/ms
함정 ④ Na⁺·K⁺ 이온 종류 혼동 5/6회 반복
정상 개념
막 투과도 그래프에서
먼저 올라가는 것 = Na⁺ (탈분극)
나중에 올라가는 것 = K⁺ (재분극)
오답 유도 내용
㉠·㉡이 어느 이온인지 확정 안 한 채로 "㉠이 세포 밖으로 이동한다" 판단 → 오류 가능
왜 틀렸나
이온 종류 먼저 확정 필수. Na⁺는 탈분극 시 세포 안으로 이동. K⁺는 재분극 시 세포 밖으로 이동.
함정 ⑤ 이온 통로 vs Na⁺-K⁺ 펌프 — ATP 사용 여부 4/6회 반복
정상 개념
이온 통로 = ATP 불필요 (확산, 농도 기울기)
Na⁺-K⁺ 펌프 = ATP 필요 (능동수송)
오답 유도 내용
"이온 통로를 통해 K⁺가 이동할 때 ATP가 사용된다" → 오답
왜 틀렸나
이온 통로는 농도 기울기에 따른 이동. 에너지 소모 없음. ATP는 펌프(능동수송)에만 사용.
▸ 근육 수축 파트
함정 ⑥ A대 길이가 수축 시 변한다 6/6회 반복
정상 개념
A대 길이는 수축·이완 관계없이 항상 1.6μm로 고정. I대(㉠)와 H대(㉢)만 변함.
오답 유도 내용
"근육이 수축하면 A대의 길이가 짧아진다" → 오답
왜 틀렸나
A대 = 마이오신 필라멘트 전체 길이. 수축 시 액틴이 마이오신 안으로 미끄러져 들어오지만 마이오신 자체 길이는 불변.
함정 ⑦ ⓐ·ⓑ·ⓒ와 ㉠·㉡·㉢ 대응 혼동 5/6회 반복
정상 개념
주어진 조건으로 대응관계 확정 먼저 → 그 후 계산

A대 = 2×㉡ + 2×㉢ = 1.6μm
오답 유도 내용
ⓐ~ⓒ를 "순서 없이" 제시. 대응관계 미확정 상태로 계산 → 엉뚱한 값 도출
왜 틀렸나
ⓐ~ⓒ와 ㉠~㉢의 매핑은 매번 다름. 반드시 대응 먼저 확정한 뒤 공식 적용.
함정 ⑧ 수축·이완 방향 판단 오류 4/6회 반복
정상 개념
X 길이 감소 → 수축
X 길이 증가 → 이완
㉠·㉢ 모두 같은 방향으로 변함
오답 유도 내용
t₁→t₂로 X 길이 감소 시 "근육이 이완된다" → 오답
왜 틀렸나
X 길이 감소 = 수축. ㉠(I대 절반)과 ㉢(H대 절반)이 동시에 감소하는지 확인하면 방향 판단 가능.
▸ 항상성 파트
함정 ⑨ ADH와 오줌량·삼투압 방향 혼동 6/6회 반복
정상 개념
ADH↑ → 수분 재흡수↑ → 오줌량↓, 오줌삼투압↑

혈장삼투압↑ → ADH↑
오답 유도 내용
"ADH 농도가 높을 때 오줌 생성량이 많다" → 오답

"혈장삼투압↑이면 ADH↓" → 오답
왜 틀렸나
ADH는 콩팥에서 수분 재흡수를 늘림. 오줌이 농축되고 양은 줄어듦. 선지는 방향이 정반대.
함정 ⑩ 인슐린·글루카곤 분비 세포 혼동 5/6회 반복
정상 개념
β세포 → 인슐린 (혈당↑ 시)
α세포 → 글루카곤 (혈당↓ 시)
오답 유도 내용
"α세포에서 인슐린이 분비된다" → 오답

1형 당뇨 = 수용체 이상이라 서술 → 오답
왜 틀렸나
1형 = β세포 파괴 (인슐린 분비 불가)
2형 = 수용체 이상
세포와 호르몬 대응 정확히 구분 필수.
함정 ⑪ 티록신 음성 피드백 방향 혼동 5/6회 반복
정상 개념
티록신↑ → TRH·TSH 분비 억제

갑상샘 저하증: 티록신↓ → TSH↑
갑상샘 항진증: 티록신↑ → TSH↓
오답 유도 내용
"티록신이 증가하면 TRH 분비가 촉진된다" → 오답

TSH·티록신 농도표로 환자 유형 오매칭
왜 틀렸나
음성 피드백은 억제 방향. 티록신이 높으면 TRH와 TSH 모두 억제. 반대 방향으로 착각 유도.
함정 ⑫ 교감신경 신경절 이후 신경전달물질 혼동 5/6회 반복
정상 개념
교감 이전 말단 = 아세틸콜린
교감 이후 말단 = 노르에피네프린
부교감 이전·이후 = 아세틸콜린
오답 유도 내용
"교감신경 신경절 이후 말단에서 아세틸콜린이 분비된다" → 오답
왜 틀렸나
교감신경 이후 말단만 노르에피네프린. 이전 말단은 교감도 아세틸콜린이라 혼동 유발. 이후 말단만 다름.
함정 ⑬ 저온 자극 시 피부 혈관 반응 혼동 4/6회 반복
정상 개념
저온 → 피부혈관 수축 (교감신경) → 열 발산↓
고온 → 피부혈관 확장 → 열 발산↑
오답 유도 내용
"저온 자극 시 피부 혈관이 확장된다" → 오답

"피부 혈관 수축에 부교감신경이 관여한다" → 오답
왜 틀렸나
피부 혈관 수축은 교감신경이 관여. 저온에서 열 보존을 위해 수축. 확장이 아니며, 신경 종류도 반대로 제시.
함정 ⑭ 길항 작용 vs 음성 피드백 구분 3/6회 반복
정상 개념
길항 = 두 호르몬이 반대 방향으로 작용 (인슐린↔글루카곤)
음성 피드백 = 호르몬이 자신의 분비를 억제
오답 유도 내용
그림에서 길항과 음성 피드백을 서로 바꿔서 선지 구성 → 혼동 유도
왜 틀렸나
길항은 두 호르몬의 관계, 음성 피드백은 한 호르몬이 자신을 조절하는 원리. 그림 구조 보고 판단.
▸ 탐구 방법 파트
함정 ⑮ 조작변인 vs 통제변인 혼동 4/6회 반복
정상 개념
조작변인 = 의도적으로 다르게 설정 (처리 여부, 농도)
통제변인 = 동일하게 유지 (온도, pH 등)
오답 유도 내용
온도를 고정한 실험에서 "온도는 조작변인이다" → 오답 (통제변인임)
왜 틀렸나
조작변인은 실험에서 실제로 다르게 설정한 것만 해당. 동일하게 맞춘 온도는 통제변인.

기출 출제
A. 흥분 전도·막전위 — 매회 반드시 출제, 절대 포기 불가
선택형 고배점(3.5~3.7점) + 서답형 서술 단골.
계산 실수 없도록 아래 공식 완벽 암기 필수.
핵심 공식 막전위 상태별 분류 — 반드시 암기
휴지전위 = −70 mV (자극 없을 때, 안정 상태) 탈분극 : −70 → +30 mV ← Na⁺ 세포 안으로 유입 (이온 통로, ATP 불필요) 재분극 : +30 → −70 mV ← K⁺ 세포 밖으로 유출 (이온 통로, ATP 불필요) 과분극 = −80 mV (K⁺ 과잉 유출 후, 잠시) Na⁺-K⁺ 펌프 → Na⁺ 3개 밖으로, K⁺ 2개 안으로 → ATP 필요 (능동수송)
핵심 공식 흥분 전도 속도 계산법
활동전위 그래프에서 각 단계까지 걸리는 시간: → +30 mV 도달 = 자극 후 약 1 ms → −80 mV (과분극) = 자극 후 약 2 ms → −70 mV 복귀 = 자극 후 약 3 ms 전도 속도 = 거리 ÷ (전체 경과시간 − 막전위 변화 시작까지 시간) 예) 경과 4 ms, +30 mV → 이동 시간 = 4 − 1 = 3 ms 거리 6 cm → 속도 = 6 ÷ 3 = 2 cm/ms
⚠ 시냅스가 있는 경우 시냅스 지연 시간이 추가됨.
🚨 시험에서 가장 많이 틀리는 포인트 — 반드시 암기
시냅스는 축삭돌기 말단 → 가지돌기 방향으로만 전달
역방향 전달은 절대 불가
이유
신경전달물질 수용체가 시냅스 후 뉴런(가지돌기 쪽)에만 존재.
시냅스 이전 뉴런(축삭돌기) 쪽에는 수용체 없음 → 전달 불가.
시험에 적용하는 법
막전위 표에서 어떤 지점이 흥분 전달을 못 받았으면(−70mV 유지)
→ 그 지점과 자극 지점 사이에 시냅스가 있고 자극이 역방향으로 진행된 것.
→ 또는 시냅스 이후 뉴런에 자극을 줬을 때, 이전 뉴런 방향으로는 막전위 변화 없음.

핵심 공식 각 구간 명칭과 관계식
㉠ = 액틴 필라멘트만 있는 부분 (I대 절반, Z선~겹침 경계) ㉡ = 액틴+마이오신 겹치는 부분 ㉢ = 마이오신만 있는 부분 (H대 절반) X의 길이 = 2×㉠ + 2×㉡ + 2×㉢ (좌우 대칭) A대 길이 = 2×㉡ + 2×㉢ = 항상 1.6 μm (수축해도 불변) H대 길이 = 2×㉢ I대 길이 = 2×㉠ ∴ 수축 시: ㉠↓ ㉢↓ ㉡↑ A대 불변
풀이 순서 계산 문제 접근 5단계
1
A대 = 1.6 μm 고정 → 2×㉡ + 2×㉢ = 1.6 세팅
2
주어진 X의 길이로 ㉠ 계산: ㉠ = (X − 1.6) ÷ 2
3
주어진 조건(비율, 합, 차)으로 ㉡, ㉢ 확정
4
ⓐ~ⓒ가 ㉠~㉢ 중 무엇인지 대입 후 확인
5
다른 시점: A대는 그대로 두고 ㉠·㉢만 변화시켜 재계산

개체군 = 일정 지역에 사는 같은 종의 개체들의 집단 예) 한 연못의 붕어 집단 / 한 숲의 소나무 집단 개체군 밀도 = 단위 면적(또는 부피)당 개체 수 → 먹이, 서식 공간, 기후 등에 따라 변함 생장 곡선 2가지: J자형 — 환경 저항 없을 때 (이론적, 지수 성장) → 실제 자연에서는 나타나기 어려움 S자형 — 환경 저항 있을 때 (실제 자연) → 개체 수가 늘수록 먹이·공간 부족 → 최대 개체 수(환경 수용 능력, K)에서 안정
💡 S자형에서 개체군이 가장 빠르게 증가하는 시점은 K/2(최대 개체 수의 절반) 부근.

예) 말미잘과 흰동가리 / 콩과식물과 뿌리혹박테리아(질소 고정)
편리공생 한 종만 이득(+)
다른 종 무관(0)		 한 종만 이득, 다른 종은 영향 없음.
예) 빨판상어가 상어에 붙어 이동 (상어는 이득도 손해도 없음)
기생 기생자 이득(+)
숙주 손해(−)		 한 종(기생자)이 다른 종(숙주)에 붙어 영양 착취.
예) 회충(기생자) ↔ 사람(숙주) / 겨우살이 ↔ 나무
💡 포식과 피식 vs 기생 구분: 포식은 먹어 없애는 것, 기생은 숙주를 죽이지 않고 계속 착취하는 것.
💡 경쟁 vs 포식 구분: 경쟁은 같은 자원을 두고 다투는 것(둘 다 −), 포식은 한 종이 다른 종을 먹는 것(+ / −).
개념 5 에너지 흐름과 물질 순환
에너지 흐름: 태양 → 생산자(광합성) → 1차소비자 → 2차소비자 → 3차소비자 → 각 단계에서 열에너지로 손실 → 단방향, 순환하지 않음 → 상위 영양단계로 갈수록 에너지양 감소 물질(탄소·질소 등) 순환: 생산자 → 소비자 → 분해자 → 비생물적 환경 → 생산자 → 순환함 (에너지와 다름) 핵심 차이: 에너지 = 단방향 흐름, 재사용 불가 물질 = 순환, 재사용 가능
💡 "에너지는 순환한다"는 오답. "물질은 순환한다"는 정답. 이 차이가 선지에 자주 나옴.
개념 6 먹이 사슬과 먹이 그물
먹이 사슬: 생산자 → 1차 소비자 → 2차 소비자 → 3차 소비자 (선형, 단순) 예) 풀 → 메뚜기 → 개구리 → 뱀 → 매 먹이 그물: 여러 먹이 사슬이 복잡하게 연결된 것 → 생태계가 안정적으로 유지되는 이유 → 한 종이 사라져도 다른 경로로 에너지 흐름 가능 영양단계: 생산자 = 1영양단계 1차 소비자 = 2영양단계 2차 소비자 = 3영양단계
💡 화살표 방향 = 에너지(유기물)가 이동하는 방향 = 먹히는 쪽 → 먹는 쪽. 즉 풀 → 메뚜기에서 화살표는 메뚜기가 풀을 먹는다는 의미.
