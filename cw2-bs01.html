Ⅰ. 생명 시스템의 구성
1. 생물과 생명과학의 특성 — 생물의 특성
⚠️ 출제 포인트
생물의 특성 6가지는 예시와 함께 연결해서 암기. "이 예시는 어떤 특성인가?" 형태로 출제됨. 바이러스의 생물적/비생물적 특성 구분도 매우 자주 출제.
① 생물의 특성 6가지 ★★★ 필수암기
특성핵심 설명예시 (암기)
세포로 구성 모든 생물은 세포로 구성. 세포 = 구조적 단위 + 기능적 단위 단세포: 아메바, 짚신벌레 / 다세포: 참나무, 다람쥐
물질대사 생명체에서 일어나는 화학 반응. 효소가 촉매로 작용. 동화작용 + 이화작용으로 구분 광합성(동화), 세포호흡(이화), 단백질 합성, 머리카락 자랄 때 아미노산 결합→단백질 합성
자극에 대한 반응과 항상성 환경 변화를 자극으로 받아들이고 반응. 체내 상태를 일정하게 유지(항상성) 가시에 손이 찔리면 손을 뗌(반응), 식물이 빛을 향해 자람(굴광성), 더울 때 땀 흘림(항상성), 물 많이 마시면 오줌량 증가(항상성)
발생과 성장 수정란 → 세포분열 → 세포 수 늘고 구조·기능 다양해지는 발생 과정 → 하나의 개체. 이후 세포분열로 세포 수 늘고 몸집이 커지고 무게 증가하는 성장 개구리 수정란→올챙이→개구리, 잠자리 알낳음→성체
생식과 유전 생식으로 자신과 닮은 자손 만들어 종족 유지. 어버이의 형질이 자손에게 전달됨(유전) 강아지가 어미를 닮음, 고양이 새끼가 어미를 닮음
적응과 진화 환경과 상호작용하면서 구조·기능·생활습성이 변하는 것 = 적응. 오랜 세월 환경 변화에 적응하면서 유전자가 변화하여 생물의 구조·기능이 변하거나 새로운 종이 나타나는 것 = 진화 선인장 잎→가시(물 손실 최소화), 가랑잎벌레(주변 잎과 비슷한 형태로 보호), 북극여우(몸집 크고 귀 작음) vs 사막여우(몸집 작고 귀 큼)
② 바이러스의 특성 ★★★ 매우 자주 출제
💡 핵심 — 생물적 특성 + 비생물적 특성 동시 보유
구분내용
비생물적 특성 • 세포 구조 없음 (세포막·핵 없음)
• 생명체 밖에서는 단백질 입자로 존재
• 스스로 물질대사 불가
• 스스로 증식 불가
생물적 특성 • 유전물질(DNA 또는 RNA) 보유
• 살아있는 숙주세포 내에서 숙주의 효소를 이용해 증식 가능
• 증식 과정에서 돌연변이 발생 → 적응과 진화 일어남

박테리오파지(T4): 세균을 숙주로 하는 바이러스. 머리(DNA)+단백질 껍질+꼬리로 구성. 세균 밖에서는 입자로 존재하여 물질대사 불가. 세균에 DNA를 주입한 후 세균에 있는 효소를 이용해 증식. 이때 증식 과정에서 돌연변이 발생 가능

⚠️ 선지 함정 정리
  • "바이러스는 세포로 구성된다" → 틀림
  • "바이러스는 유전물질을 가진다" → 맞음
  • "바이러스는 스스로 물질대사를 한다" → 틀림 (숙주세포 내에서만 가능)
  • "바이러스는 적응과 진화가 일어난다" → 맞음 (증식 과정에서 돌연변이 발생)
  • "바이러스는 세포 밖에서 증식할 수 없다" → 맞음
2. 생명과학의 특성
핵심 내용
  • 생명과학 = 지에 살고 있는 생물의 특성과 다양한 생명 현상을 연구하는 과학의 한 분야
  • 생명의 기원, 생물의 구조와 기능, 생물의 생식과 유전, 생물의 분류 및 분포 등을 연구
  • 세부 학문 분야: 세포학, 분류학, 생태학, 생리학, 유전학, 발생학, 생화학, 분자생물학, 신경생물학, 생명공학
  • 생명과학의 성과는 여러 학문 분야의 성과와 결합되어 나타남 (화학, 물리학, 컴퓨터과학, 수학, 법학 등)
  • 생물정보학: 컴퓨터를 이용해 DNA 염기서열, 단백질 아미노산서열 분석하여 유전자 변환이나 단백질 구조 연구
  • 생물지리학: 생물의 분포와 그것에 영향을 미치는 자연환경과의 관계 연구
  • 법의학: 유전자 검사를 통한 개인 식별, 사망 원인 등 법률상 문제가 되는 의학적 사항에 대한 증거 제공
  • 사례 1: 흑등고래 지느러미 구조를 풍력발전기 회전 날개에 적용 → 효율 증가 (생명과학 + 공학)
  • 사례 2: 바이오디젤, 바이오에탄올을 생산하는 미세조류를 대량 배양 (생명과학 + 화학·공학)
  • 사례 3: 사건 현장의 혈액·머리카락에서 DNA 채취 → 범인 신원 확인 (생명과학 + 법학)
3. 생명 시스템의 구성 단계
⚠️ 핵심 출제 포인트
동물과 식물의 구성 단계 차이를 정확히 구분. "조직계"는 식물에만 있고, "기관계"는 동물에만 있음!
① 동물의 구성 단계 ★★★
세포 → 조직 → 기관 → 기관계 → 개체 → 생태계
단계설명예시
세포구조적·기능적 단위. 세포소기관으로 구성근육세포, 신경세포
조직모양·기능이 비슷한 세포들의 모임근육조직, 신경조직
기관여러 조직이 모여 고유한 형태와 기능을 가짐위, 심장, 폐, 콩팥
기관계여러 기관이 모여 독립적으로 생명활동 수행소화계, 순환계, 호흡계, 배설계
개체독립적으로 생명활동을 하는 하나의 생물체사람, 다람쥐
② 식물의 구성 단계 ★★★
세포 → 조직 → 조직계 → 기관 → 개체
  • 식물에는 기관계가 없고 조직계가 있음
  • 조직계: 표피조직계, 관다발조직계, 기본조직계로 구분
  • 식물 기관: 뿌리, 줄기, 잎, 꽃, 열매

생명 시스템 개념
  • 시스템: 구성요소들이 상호작용하거나 상호 의존하여 복잡하게 얽힌 통일된 하나의 집합체
  • 세포, 개체, 생태계는 각각 하나의 생명 시스템
  • 생태계: 생물들이 다른 생물 및 빛, 온도, 물, 토양, 공기 등의 환경과 영향을 주고받으며 살아가는 체계
  • 생태계는 숲, 사막, 연못, 하천, 대양 등 다양하며, 경작지·저수지·공원 등 인위적으로 만들어진 것도 생태계라 할 수 있음
4. 물질대사 — 생명활동과 에너지
① 물질대사의 구분 ★★★
구분동화작용이화작용
방향작은 분자 → 크고 복잡한 분자크고 복잡한 분자 → 작은 분자
에너지에너지 흡수 (흡열반응)에너지 방출 (발열반응)
예시광합성: CO₂+H₂O→포도당+O₂
단백질 합성: 아미노산→단백질		세포호흡: 포도당+O₂→CO₂+H₂O+에너지
단백질 분해: 단백질→아미노산
💡 효소와 촉매
  • 물질대사는 반드시 효소(촉매)가 관여
  • 효소: 반응 전후에 소모되거나 변하지 않음. 활성화에너지를 낮춰 물질대사를 촉진
  • 생물이 가진 촉매 → 효소 (단백질로 구성)
  • 생물 밖에서 포도당이 연소되려면 400℃ 이상 필요. 생물 몸 안에서는 체온 정도에서 분해 가능 → 효소 덕분
  • 효소는 특정 기질에만 작용 (기질 특이성)
  • 효소의 활성: 온도·pH의 영향을 받음
② ATP 구조와 역할 ★★★ 완전 암기
ATP 구조

ATP = 아데닌 + 라이보스 + 인산 3개가 결합한 화합물 (아데노신 3인산)

인산과 인산 사이의 결합(~)이 끊어지면 다량의 에너지 방출

ATP → ADP + 무기인산(P) + 에너지 (에너지 방출, 생명활동에 사용) ADP + 무기인산(P) + 에너지 → ATP (에너지 저장, 세포호흡에서 방출된 에너지 이용)

ATP: 생명활동에 직접 사용되는 에너지원

③ 세포호흡과 에너지 전환 ★★★
세포호흡
  • 세포에서 포도당 같은 영양소의 에너지를 생명활동에 사용할 수 있는 형태(ATP)로 전환하는 과정
  • 주로 미토콘드리아에서 일어남
  • 포도당 + 산소 → 이산화탄소 + 물 + 에너지(ATP)
  • 방출된 에너지 일부 → ATP에 화학 에너지 형태로 저장
  • 나머지 에너지 → 열로 방출
ATP 에너지 전환과 이용 — 전체 흐름 암기
영양소 → 세포호흡 → 에너지 방출 → ATP 합성 ATP → ADP + P 로 분해 → 에너지 방출 → 생명활동에 이용

ATP에 저장된 에너지는 다음과 같이 전환되어 사용됨:

  • 기계적 에너지: 근육수축
  • 열에너지: 체온 유지
  • 화학 에너지: 물질 합성
  • 전기 에너지: 신경신호 전달
  • 빛에너지: 반딧불이 발광 (아랫배 끝에 있는 발광세포에서 빛 생산)
  • 전기 에너지: 전기뱀장어가 전기 생산
  • 그 외: 성장, 정신 활동, 물질 이동 등
5. 기관계의 통합적 작용
⚠️ 출제 포인트
4대 기관계의 기능을 정확히 구분하고, 서로 어떻게 연결되는지 통합적으로 이해. 각 기관계에 속하는 기관도 암기 필수. 기관계 연결 다이어그램 형태로 출제됨.
① 4대 기관계와 속하는 기관 ★★★
기관계기능속하는 기관
소화계 음식물의 영양소를 작은 영양소로 분해하여 흡수 입, 식도, 위, 작은창자, 큰창자, 간, 이자, 쓸개
호흡계 산소를 흡수하고, 세포호흡으로 생성된 이산화탄소를 배출 코, 숨관(기관), 숨관가지(기관지), 폐
순환계 영양소·산소·이산화탄소·노폐물 등의 물질을 운반 심장, 동맥, 정맥, 모세혈관
배설계 혈액을 걸러 질소 노폐물과 여분의 물을 오줌 형태로 내보냄 콩팥, 오줌관, 방광, 요도
② 영양소 소화와 흡수 ★★
영양소분해 효소분해 산물흡수 경로
녹말(탄수화물)탄수화물분해효소포도당융털 모세혈관 → 혈액
단백질단백질분해효소아미노산융털 모세혈관 → 혈액
지방지방분해효소지방산 + 모노글리세리드융털 암죽관 → 림프관
  • 소화는 소화계에서. 분해된 영양소는 대부분 작은창자에서 흡수
  • 작은창자 내벽에 많은 주름과 융털이 있어 흡수 면적이 넓음
  • 흡수된 영양소는 혈액을 따라 심장→온몸의 조직 세포로 운반됨
③ 산소 흡수와 이산화탄소 배출 ★★
  • 숨을 들이쉬면 공기가 코 → 숨관 → 숨관가지 → 폐로 이동
  • 폐의 허파꽈리에서 산소와 이산화탄소 교환 (가스교환)
  • 허파꽈리 → 모세혈관으로 산소 확산 → 혈액 → 심장 → 온몸 조직 세포

① 탄소순환 ★★★
탄소순환 경로
  • 탄소는 주로 이산화탄소(CO₂) 형태로 대기 중에 존재하거나 물에 녹아 있음
  • 대기 중 CO₂ → 생산자(광합성) → 포도당 등 유기물로 전환
  • 유기물 → 먹이사슬을 따라 생산자→소비자로 이동
  • 생산자·소비자의 세포호흡 → CO₂ 방출 → 대기
  • 생물의 사체·배설물 속 유기물 → 분해자가 분해 → CO₂ 형태로 대기 또는 물속으로
  • 생물의 사체 중 분해되지 않은 유기물 → 땅속·호수·해저에 오랜 기간 퇴적 → 화석연료로 변화 → 연소 → CO₂ 형태로 다시 대기로
  • 화석연료 사용 증가 → 대기 중 CO₂ 농도 증가 → 지구 온난화 심화
② 질소순환 ★★
질소순환 경로
  • 대기 중 질소(N₂)는 대부분의 생물이 직접 이용 불가
  • 질소고정: 뿌리혹세균(콩과식물의 뿌리혹), 아조토박터 등 질소고정세균이 대기 중 N₂를 암모늄 이온(NH₄⁺)으로 전환. 공중 방전(번개)으로도 질소 화합물 형성됨
  • 질산화 작용: 질산화세균이 암모늄 이온(NH₄⁺) → 질산 이온(NO₃⁻)으로 전환 → 식물이 뿌리로 흡수하여 이용
  • 질소 화합물 → 먹이사슬을 따라 생산자→소비자로 이동
  • 사체·배설물 속 질소 화합물 → 분해자에 의해 암모늄 이온으로 분해
  • 암모늄 이온 → 질산화세균에 의해 다시 질산 이온으로 전환 → 식물이 이용
  • 탈질산화 작용: 탈질산화세균이 토양 속 질산 이온(NO₃⁻)을 질소(N₂)로 환원 → 대기로 돌아감
⚠️ 질소순환 선지 함정
  • "질소고정 과정에는 뿌리혹세균이 관여한다" → 맞음
  • "탈질산화 과정은 분해자에 의해 일어난다" → 틀림 (탈질산화세균이 관여)
  • "식물은 뿌리를 통해 질산 이온이나 암모늄 이온을 흡수하여 질소 화합물 합성에 이용한다" → 맞음
  • "탈질산화 작용에는 질산화세균이 관여한다" → 틀림 (탈질산화세균)
9. 에너지흐름 — 생태피라미드, 생태계평형
① 에너지흐름 ★★★
에너지흐름 원리
  • 태양에서 오는 빛에너지 → 생산자의 광합성 → 유기물 속 화학 에너지로 전환
  • 먹이사슬을 따라 상위 영양단계로 이동
  • 각 영양단계에서 에너지의 일부는 호흡을 통해 열에너지로 방출 → 생태계 밖으로 빠져나감
  • 일부 에너지가 다음 영양단계로 전달됨
  • 생물의 사체나 배설물 속 에너지 → 분해자의 호흡 → 열에너지로 방출
  • 생태계에서 에너지는 순환하지 않고 한 방향으로 이동하며 생태계 밖으로 빠져나가므로 생태계가 유지되려면 끊임없이 태양에서 에너지가 공급되어야 함
  • 상위 영양단계로 갈수록 사용할 수 있는 에너지양이 점점 줄어듦 → 먹이사슬의 영양단계는 일반적으로 몇 단계로 제한됨
② 물질의 생산과 소비 ★★ 빠졌던 내용

용어설명

피식량순생산량 중 초식동물에게 먹히는 양

③ 에너지효율 계산 ★★★ 계산 필수
🔢 에너지효율 공식과 계산 방법
에너지효율(%) = (현 영양단계가 보유한 에너지 총량 / 전 영양단계가 보유한 에너지 총량) × 100

일반적으로 에너지효율 = 5% ~ 20%

예시 계산 (교과서 수치):

  • 태양의 빛에너지: 20810 kcal/m²/yr
  • 생산자 에너지: 20810 → 1차 소비자: 3368 → 2차 소비자: 383 → 3차 소비자: 21 (단위: kcal/m²/yr)
  • 분해자에게 가는 에너지: 사체 및 배설물에서 1095(1차소비자→분해자), 46, 6
  • 1차 소비자 에너지효율 = 3368/20810 × 100 ≈ 16.2%
  • 2차 소비자 에너지효율 = 383/3368 × 100 ≈ 11.4%
  • 3차 소비자 에너지효율 = 21/383 × 100 ≈ 5.5%

풀이 순서:

  1. ① 현 영양단계의 에너지 총량 확인 (호흡량 + 피식량 + 사체배설물 포함 전체)
  2. ② 전 영양단계의 에너지 총량 확인
  3. ③ (현/전) × 100 계산
④ 생태피라미드와 생태계평형 ★★★
생태피라미드
  • 생태계에서 하위 영양단계부터 상위 영양단계까지 각 영양단계의 에너지양, 개체 수, 생체량(현재 생물군집이 가지고 있는 유기물의 총량)을 차례로 쌓아 올린 것
  • 안정된 생태계에서는 상위 영양단계로 갈수록 에너지양·개체 수·생체량이 줄어듦 → 피라미드 모양
  • 에너지피라미드, 개체수피라미드, 생체량피라미드가 있음
생태계평형
  • 생태계에서 생물군집의 구성이나 개체 수, 물질의 양, 에너지흐름이 안정된 상태를 유지하는 것
  • 주로 먹이사슬에 의해 유지됨. 먹이사슬이 복잡할수록 생태계평형이 잘 유지됨
  • 생태계평형 회복 과정: 생산자·소비자·분해자의 물질 생산과 소비의 균형이 이루어지고, 먹이사슬에 따른 에너지흐름이 원활하게 이루어지면 생태계가 안정적으로 유지됨
  • 1차 소비자 일시적 증가 → 생산자 감소 + 2차 소비자 증가 → 1차 소비자 감소 → 생산자 회복 → 평형 회복
  • 생태계평형을 깨는 요인: 외래생물 유입(큰입배스), 댐 건설, 화석연료 과다 사용, 해양 유류 오염 사고
  • 생태계가 평형을 유지하는 능력에는 한계가 있으며, 이 한계를 넘는 요인이 작용하면 평형이 깨짐. 회복하기까지 오랜 시간이 걸리며 때로는 회복하지 못하기도 함

유형특징예시
Ⅰ형 초기 사망률 낮고 대부분 성체로 성장. 수명 길다. 초기 사망률 낮고 자손 수 적음 기린, 사람, 코끼리 등 대형 포유류 (자손 수 적지만 오래 보살핌)
Ⅱ형 연령에 관계없이 사망률이 일정. 초기 사망률 낮고 수명이 길어 대부분 성체로 성장 아프리카코끼리, 돌산양
Ⅲ형 초기 사망률이 매우 높아 성체로 성장하는 개체 수 적다. 알을 많이 낳음 굴, 어류 (알을 많이 낳지만 초기에 많이 죽음)
💡 초기 사망률과 자손의 수·수명의 관계

초기 사망률 낮음 + 자손 수 적음 + 수명 길다 → Ⅰ형 (사람, 대형 포유류)

초기 사망률 높음 + 자손 수 많음 → Ⅲ형 (어류, 굴)

→ 초기 사망률이 낮고 대부분 성체로 성장하는 생물 개체군은 Ⅰ형 생존곡선을 나타냄

→ 많은 수의 알을 낳는 어류에서 주로 Ⅲ형 생존곡선이 나타남

④ 개체군의 주기적 변동 ★
계절에 따른 돌말 개체군의 주기적 변동
  • 자연 상태에서 개체군의 크기는 환경 요인의 변화에 따라 주기적으로 변동함
  • 호수나 유속이 느린 하천에서 사는 돌말: 빛의 세기·수온·영양염류의 양 등 계절적 변화에 따라 개체군의 크기가 주기적으로 변동
  • 봄: 빛의 세기 강해지고 수온 높아짐 → 개체 수 급격히 늘어남
  • 여름: 영양염류의 양이 고갈됨 → 개체 수 줄어듦
  • 가을: 영양염류의 양이 많아짐 → 개체 수 다시 늘어남
  • 겨울: 빛의 세기 약해지고 수온 낮아짐 → 개체 수 다시 줄어듦

② 군집의 구조 — 층상구조와 우점종 ★★
삼림의 층상구조

삼림과 같이 많은 개체군으로 구성된 군집은 수직적인 몇 개의 층으로 구성된 층상구조를 이룸

위에서부터: 교목층 → 아교목층 → 관목층 → 초본층 → 선태층 → 지중층

삼림의 교목층과 아교목층에는 그 군집을 대표할 만한 수목들이 자람

우점종: 군집에서 가장 수가 많거나 가장 넓은 면적을 차지하여 군집을 대표할 수 있는 개체군

삼림의 이름은 우점종의 이름을 따서 붙임 (신갈나무 우점 → 신갈나무림, 소나무 우점 → 소나무림)

③ 군집 조사 방법 — 방형구법 ★★★ 계산 필수
🔢 방형구법 공식 — 완전 암기
밀도 = 특정 종의 개체 수 / 전체 방형구의 면적(m²) 빈도 = 특정 종이 출현한 방형구 수 / 전체 방형구 수 피도 = 특정 종의 점유 면적(m²) / 전체 방형구의 면적(m²) 상대밀도(%) = (특정 종의 밀도 / 조사한 모든 종의 밀도의 합) × 100 상대빈도(%) = (특정 종의 빈도 / 조사한 모든 종의 빈도의 합) × 100 상대피도(%) = (특정 종의 피도 / 조사한 모든 종의 피도의 합) × 100 중요치 = 상대밀도 + 상대빈도 + 상대피도 → 중요치가 가장 높은 종 = 우점종

예시 계산: 방형구 4개(각 1m×1m=1m²), 전체 면적 4m²

  • 민들레: 개체 수 8, 출현 방형구 수 3, 차지한 면적 0.6m²
  • 밀도 = 8/4 = 2 / 빈도 = 3/4 = 0.75 / 피도 = 0.6/4 = 0.15
  • 이런 식으로 모든 종 계산 후, 상대값 구하고 중요치 비교하여 우점종 결정
⚠️ 방형구 계수 주의
  • 방형구 경계에 걸쳐 있는 식물: 중심부가 방형구 안에 있는 것만 개체 수에 포함
  • 방형구 한 칸(2cm×2cm)에 2개의 식물이 출현한 경우 면적을 1/2씩 차지하는 것으로 함

  • 말이집: 슈반세포의 세포막이 축삭을 여러 겹 싸고 있는 구조. 신경자극이 이동할 때 절연체 역할
  • 랑비에결절: 말이집과 말이집 사이에 축삭이 노출되어 있는 부분

13. 신경자극의 전도 — 막전위, 활동전위
① 막전위와 이온 분포 — 휴지 상태 ★★★ 완전 암기
💡 휴지 상태(분극) 이온 분포 — 절대 혼동 금지
이온세포 안세포 밖
K⁺더 많음적음
Na⁺적음더 많음
  • Na⁺-K⁺펌프(능동수송, ATP 소모): Na⁺을 세포 밖으로 내보내고, K⁺을 세포 안으로 이동시킴
  • K⁺통로: 일부 열려 있어 K⁺이 세포 안→밖으로 확산
  • Na⁺통로: 거의 닫혀 있어 Na⁺가 세포 안으로 확산 어려움
  • 결과: 세포 안 = 음(−)전하, 세포 밖 = 양(+)전하 → 분극 상태
  • 휴지전위: 일반적으로 −70mV ~ −80mV

💡 역치와 실무율
  • 역치: 반응을 일으킬 수 있는 최소한의 자극 세기
  • 역치 미만 자극 → 활동전위 발생하지 않음
  • 역치 이상 자극 → 실무율(all-or-none): 자극 세기에 관계없이 활동전위의 크기는 항상 일정
  • 즉, 역치 이상이면 자극이 강해도 약해도 동일한 크기의 활동전위 발생
③ 신경자극의 전도 과정 ★★★
전도 과정
  • 뉴런의 막 한 부위에서 Na⁺이 세포 안으로 들어와 활동전위가 발생하면, 이웃한 부위에서 연속적으로 탈분극이 일어나 새로운 활동전위가 발생하고, 활동전위가 발생했던 부위에서는 K⁺이 세포 밖으로 나가 재분극이 일어남
  • 이와 같이 활동전위가 축삭돌기를 따라 연속적으로 발생하면서 신경자극이 전도됨
  • 신경자극이 축삭돌기 말단까지 이동하는 동안 연속적으로 발생하는 활동전위의 크기는 일정함
  • 전도 속도: 말이집신경 > 민말이집신경

④ 신경자극 전도 속도 계산 ★★★ 서술형 단골
🔢 전도 속도 계산 — 완전 정복
전도 속도(cm/ms) = 두 지점 사이의 거리(cm) / 두 지점에서 활동전위 발생 시간 차이(ms) 또는 특정 지점의 활동전위 발생 시간 = 자극 준 시점 + (자극 지점에서 측정 지점까지의 거리 / 전도 속도)

예시 문제 풀이 (교과서 기준):

민말이집신경의 네 지점 d₁~d₄, 역치 이상 자극을 d₁에 줌. 전도 속도 1cm/ms. d₁=0cm, d₂=1cm, d₃=2cm, d₄=3cm

  • d₁에 자극: 0ms에 활동전위 발생
  • d₂ 도달 시간: 1cm ÷ 1cm/ms = 1ms → d₂는 1ms에 활동전위 발생
  • d₃: 2cm ÷ 1cm/ms = 2ms → d₃는 2ms에 활동전위 발생
  • d₄: 3cm ÷ 1cm/ms = 3ms → d₄는 3ms에 활동전위 발생

자극 후 4ms가 경과했을 때 각 지점의 막전위 상태:

  • d₁ (0cm): 4ms 경과 → 활동전위 발생(0ms) 후 4ms 지남 → 분극 상태 회복 (−70mV)
  • d₂ (1cm): 1ms에 활동전위 발생 후 3ms 지남 → 분극 상태 회복 (−70mV)
  • d₃ (2cm): 2ms에 활동전위 발생 후 2ms 지남 → 재분극 or 분극 회복 중 (−70mV 근처 또는 이하)
  • d₄ (3cm): 3ms에 활동전위 발생 후 1ms 지남 → 재분극 중
⚠️ 계산 풀이 핵심 순서
  1. ① 자극 지점에서 측정 지점까지의 거리 확인
  2. ② 전도 속도로 이동 시간 계산 (시간=거리/속도)
  3. ③ 자극 시점 + 이동 시간 = 활동전위 발생 시점 계산
  4. ④ 문제에서 주어진 시간에서 활동전위 발생 시점을 뺀 경과 시간 계산
  5. ⑤ 경과 시간에 따라 막전위 상태(탈분극/재분극/분극) 판단
  6. ⑥ 이온 이동 방향(Na⁺ 방향: 탈분극 시 밖→안, K⁺ 방향: 재분극 시 안→밖) 결정
14. 시냅스전달
① 시냅스의 구조 ★★★
시냅스 구조
  • 한 뉴런의 축삭돌기 말단은 다른 뉴런과 약 20~40nm의 좁은 틈을 두고 접해 있음
  • 이 접속 부위 = 시냅스
  • 시냅스에 있는 틈 = 시냅스틈
  • 시냅스전뉴런: 신호를 보내는 쪽 뉴런 (축삭돌기 말단에 시냅스소포 있음)
  • 시냅스소포: 신경전달물질이 들어 있음. 축삭돌기 말단에 위치
  • 시냅스후뉴런: 신호를 받는 쪽 뉴런 (세포막에 수용체 있음)
② 시냅스전달 과정 ★★★ 순서 암기
💡 시냅스전달 과정 — 순서대로 암기
  1. 신경자극이 시냅스전뉴런의 축삭돌기 말단에 도달
  2. 시냅스소포가 세포막과 융합
  3. 신경전달물질이 시냅스틈으로 방출
  4. 신경전달물질이 시냅스후뉴런의 세포막에 있는 수용체에 결합
  5. 이온 통로가 열려 Na⁺가 시냅스후뉴런의 세포 안으로 유입
  6. 시냅스후뉴런 탈분극 발생 → 신호 전달 완료
⚠️ 시냅스 핵심 특징 — 시험 단골 함정
  • 시냅스전달은 시냅스전뉴런 → 시냅스후뉴런 방향으로만 일어남. 역방향 불가
  • 이유: 신경전달물질이 들어 있는 시냅스소포는 뉴런의 축삭돌기 말단에만 있고, 신경전달물질의 수용체는 뉴런의 신경세포체나 가지돌기에 있어 시냅스전달은 시냅스전뉴런에서 시냅스후뉴런 쪽의 한 방향으로만 일어나며, 반대 방향으로는 일어나지 않음
  • 한 뉴런 내에서는 전기적 신호(활동전위)로 전달 → 빠름
  • 시냅스에서는 화학 물질(신경전달물질)로 전달 → 약간 느림
③ 신경전달물질의 종류 ★★

20. 항상성 유지 — 혈장 삼투압 조절 (ADH)
ADH 조절 메커니즘 ★★★ 매년 그래프로 출제
💡 삼투압이란?

반투과성막을 사이에 두고 농도가 다른 두 용액이 있을 때 농도가 낮은 쪽에서 높은 쪽으로 물이 이동하는 현상을 삼투라 하며, 이때 나타나는 압력을 삼투압이라 함

체액의 농도가 높아지면 혈장 삼투압이 높아지고, 낮아지면 혈장 삼투압이 낮아짐

상황ADH 분비량콩팥에서 물 재흡수오줌량혈장 삼투압 변화
혈장 삼투압 높을 때
(땀 많이 흘림, 짠 음식 섭취)		 증가 증가 감소 (오줌 농도 증가) 낮아짐 → 정상 회복
혈장 삼투압 낮을 때
(물 많이 마심)		 감소 감소 증가 (오줌 농도 감소) 높아짐 → 정상 회복
⚠️ ADH 핵심 사항 — 절대 암기
  • ADH 합성 장소: 시상하부
  • ADH 분비 장소: 뇌하수체 후엽
  • ADH 조절 중추: 시상하부
  • ADH 작용 표적기관: 콩팥 (콩팥에서 물의 재흡수 촉진)
  • "ADH 분비가 증가하면 오줌 생성량이 증가한다" → 틀림 (오줌량 감소)
  • "ADH 분비가 증가하면 혈장 삼투압이 감소한다" → 맞음
  • 정상 상태일 때 콩팥에서 물의 재흡수량 < 혈장 삼투압 변화 시 물의 재흡수량 (ADH 분비 증가 시)
🔢 ADH 그래프 해석 — 시험 단골

혈장 삼투압 변화(%)-ADH 농도 그래프 해석:

  • 안정 상태(혈장 삼투압 정상)일 때 ADH 농도 = 기준값
  • 혈장 삼투압 증가(+방향) → ADH 농도 증가 (그래프에서 오른쪽으로 갈수록 ADH ↑)
  • 혈장 삼투압 감소(−방향) → ADH 농도 감소 또는 0에 가까워짐
  • ADH 농도 높음 → 콩팥에서 물 재흡수량 많음 → 오줌 생성량 적음
  • 안정 상태일 때 콩팥에서 물의 재흡수량/물의 재흡수량(P₁일 때) < 1 → 안정 상태가 ADH 적게 분비되므로 재흡수량 적음
